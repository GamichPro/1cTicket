# 1cTicket
1С Отчеты аналиткика

## 🚀 Технологии

### 🎯 Основной фреймворк
- **⚡ Next.js 16** - React фреймворк с App Router
- **📘 TypeScript 5** - Типобезопасный JavaScript
- **🎨 Tailwind CSS 4** - Utility-first CSS

### 🧩 UI компоненты
- **🧩 shadcn/ui** - Качественные компоненты на Radix UI
- **🎯 Lucide React** - Иконки
- **🌈 Framer Motion** - Анимации
- **🎨 Next Themes** - Темная тема

### 🗄️ База данных
- **🗄️ Prisma** - TypeScript ORM
- **🔐 NextAuth.js** - Аутентификация

## 🚀 Быстрый старт

```bash
# Установка зависимостей
bun install

# Запуск разработки
bun run dev

# Запуск тестов
bun run test

# Сборка для продакшена
bun run build
```

Откройте [http://localhost:3000](http://localhost:3000)

## 📁 Структура проекта

```
src/
├── app/                 # Next.js App Router
├── components/          # React компоненты
│   └── ui/             # shadcn/ui компоненты
├── hooks/              # React хуки

└── __tests__/          # Тесты
```
